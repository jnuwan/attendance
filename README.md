# Attendance

This repository contains a Java Swing application for managing attendance. This application is a demo project showcasing basic attendance tracking functionalities.

## Features

- Add new attendees
- Mark attendance for attendees
- View attendance records
- Edit and delete attendance records

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- An IDE or text editor (e.g., IntelliJ IDEA, Eclipse, VS Code)

### Installation

1. **Clone the repository:**

   ```sh
   git clone https://github.com/jnuwan/attendance.git
   cd attendance
   ```

2. **Open the project in your preferred IDE.**

## Usage

1. **Run the application:**

   - Locate the main class (usually `Main.java` or similar).
   - Run the main class to start the application.

2. **Using the application:**

   - Use the graphical user interface (GUI) to add, edit, or delete attendees.
   - Mark attendance for each attendee.
   - View the attendance records as needed.
